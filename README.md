# KotoMusic Beta - Публичный музыкальный плеер

KotoMusic (Beta) - это современный веб-музыкальный плеер, созданный как альтернатива таким сервисам как Spotify. Проект предоставляет платформу для пользователей, чтобы они могли делиться своей музыкой и слушать треки, загруженные другими участниками.

## Особенности

- **Пользовательская система**: Регистрация, авторизация и управление профилями
- **Загрузка музыки**: Возможность для одобренных пользователей загружать свои треки
- **Модерация контента**: Система премодерации для обеспечения качества контента
- **Проигрыватель**: Полнофункциональный аудиоплеер с визуализацией, прогресс-баром и управлением
- **Административная панель**: Инструменты для модерации пользователей и контента
- **Локальное хранилище**: Использование localStorage для хранения данных между сеансами

## Архитектура

- HTML5, CSS3 и JavaScript (без внешних зависимостей)
- Использование localStorage для хранения пользовательских данных и треков
- Адаптивный дизайн, оптимизированный для различных устройств
- Современные стили с градиентами и анимациями

## Система пользователей

- Гости могут слушать одобренные треки
- Зарегистрированные пользователи ожидают одобрения
- Одобренные пользователи могут загружать треки
- Администраторы и суперадминистраторы имеют права модерации

## Технические возможности

- Воспроизведение, пауза, следующий/предыдущий трек
- Перемешивание и повтор треков
- Регулировка громкости
- Визуализация звука
- Отображение информации о треке (название, исполнитель, загрузчик)

## Демонстрация

Проект запускается как standalone HTML файл, который можно открыть прямо в браузере без серверной части.
