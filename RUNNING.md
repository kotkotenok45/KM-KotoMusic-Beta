# Запуск KotoMusic Beta

## Быстрый запуск

Для быстрого запуска приложения выполните:

```bash
./start.sh
```

Или используйте npm:

```bash
npm start
```

После запуска откройте в браузере: http://localhost:3000

## Альтернативные способы запуска

### Используя npx serve (без сервера Node.js):

```bash
npx serve .
```

### Используя nodemon для разработки:

```bash
npm run dev
```
